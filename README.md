# 📦 MyProjects Monorepo

这是一个用于管理我所有前端/React 项目的 monorepo 仓库。

## 📁 项目目录

- `project-a/` 👉 React 项目 A
- `project-b/` 👉 React 项目 B
- `project-c/` 👉 React 项目 C

## 📦 使用方式

每个子项目是独立的 React 项目，拥有自己的：

- `package.json`
- `node_modules/`
- `src/`
- `public/`

### 📦 安装依赖

进入子项目目录：

```bash
cd project-a
npm install
```

npm run dev
